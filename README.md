Описание проекта

Проект представляет собой транслятор простого языка присваиваний целочисленных переменных. 
Реализованы:
- Лексический анализ (сканер)
- Синтаксический анализ (рекурсивный спуск)
- Семантический анализ

Структура проекта

- `tokens.py` - описание типов токенов и класса Token
- `scanner.py` - лексический анализатор
- `parser.py` - синтаксический анализатор (рекурсивный спуск)
- `semantic.py` - семантический анализатор
- `main.py` - точка входа и набор тестов

Грамматика языка
<Program> ::= <StatementList>
<StatementList> ::= <Statement> | <StatementList> <Statement>
<Statement> ::= <Declaration> | <Assignment>
<Declaration> ::= int ID = <Expression> ; | int ID ;
<Assignment> ::= ID = <Expression> ;
<Expression> ::= <Term> { (+|-) <Term> }
<Term> ::= <Factor> { (*|/) <Factor> }
<Factor> ::= IntLiteral | ID | ( <Expression> ) | - <Factor>


## Установка и запуск

Обработка ошибок
Транслятор обрабатывает три типа ошибок:
Лексические ошибки - неизвестные символы в исходном коде
Синтаксические ошибки - нарушение правил грамматики
Семантические ошибки - использование необъявленных переменных,
повторное объявление, деление на ноль
Набор тестов
Проект включает тесты для проверки:
Простых объявлений и инициализаций
Приоритета операций
Множественных операций
Унарного минуса
Вложенных скобок
Цепочек вычислений
Обработки всех типов ошибок
