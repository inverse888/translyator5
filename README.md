Описание проекта

Проект представляет собой транслятор простого языка присваиваний целочисленных переменных. 
Реализованы:
- Лексический анализ (сканер)
- Синтаксический анализ (рекурсивный спуск)
- Семантический анализ

Структура проекта

- `tokens.py` - описание типов токенов и класса Token
- `scanner.py` - лексический анализатор
- `parser.py` - синтаксический анализатор (рекурсивный спуск)
- `semantic.py` - семантический анализатор
- `main.py` - точка входа и набор тестов
